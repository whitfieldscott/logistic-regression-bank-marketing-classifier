# Logistic Regression Bank Marketing Classifier

## 📌 Project Overview

This project uses a Logistic Regression model to predict whether a customer will subscribe to a term deposit based on a bank marketing campaign dataset. The goal is to support data-driven marketing decisions by identifying high-probability customers.

---

## 🎯 Business Problem

Marketing campaigns often target large audiences with low conversion rates. This leads to wasted time, effort, and cost.

This project addresses that problem by building a predictive model that helps:

* Identify customers likely to subscribe
* Improve campaign efficiency
* Reduce unnecessary outreach

---

## 🧠 Solution Approach

A Logistic Regression classification model was developed to predict customer subscription behavior.

### Workflow:

* Data loading and exploration
* Data cleaning and preprocessing
* Feature analysis
* Model training
* Model evaluation

---

## 📊 Dataset

The dataset comes from a Portuguese bank’s direct marketing campaign.

### Features include:

* Demographics (age, job, marital status, education)
* Financial indicators (housing loan, personal loan)
* Campaign data (contact type, duration, number of contacts)
* Previous campaign results

### Target:

* `y` → subscription to term deposit (yes/no)
